# Clubly

Plataforma de manejo de Clubes Deportivos de Voleibol con backend en Node.js/TypeScript y frontend en React/TypeScript.

## Estructura

- backend/: API REST con autenticacion JWT y MySQL
- frontend/: SPA en React + Vite + Tailwind
- schema.sql: esquema de base de datos MySQL
- start.ps1: script para abrir backend y frontend

## Instrucciones

- Crea los .env en base a los env.example
- Configura la carpeta base en el start.ps1
- Ejecuta en la carpeta Cubly: powershell -ExecutionPolicy Bypass -File start.ps1
